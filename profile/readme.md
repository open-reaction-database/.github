# The Open Reaction Database

FRONT PAGE README FOR THE ORD IN DEVELOPMENT....

[ord-schema](https://github.com/open-reaction-database/ord-schema) is the repository for the data schema, and associated tools for working with ORD dataset files. All ORD users need to install this package.

[ord-data](https://github.com/open-reaction-database/ord-data) is the repository where the ORD data files are stored. Go here to download individual datasets, or to upload new datasets.

[ord-interface](https://github.com/open-reaction-database/ord-interface) is the repository for the online data browser and search websites. Most users do not need to install this code, but the issues page can be used to report problems with the ORD website.

[ord-app](https://github.com/open-reaction-database/ord-app) is the repository for the online reaction editor. Most users do not need to install this code, but the issues page can be used to report problems with the ORD reaction editor.
